TK2000 II Color Computer Emulator

Ver 0.1

Requirements.

  • JDK 1.5+ (download from http://java.com/).
  • Apache Ant 1.5+ (download from http://ant.apache.org) (Required only for compilation).

Build.

run "ant" on the root folder.

Run standalone.

java -jar TK2000.jar

Run as applet.

Click here.
